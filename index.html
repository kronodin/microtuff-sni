How to Import .ehi Config Files

  1. Download and install HTTP Injector from the Google Play Store or a trusted APK source.
  2. Open HTTP Injector on your device.
  3. Tap the Menu (three dots) in the top-right corner.
  4. Select Import Config or Settings → Config → Import.
  5. Navigate to your Downloads folder and select the .ehi file.
  6. The config will appear in your config list. Tap it to activate.
  7. Press Start to establish the VPN connection.

Troubleshooting Tips

  • Ensure your mobile data is connected (WiFi may block VPN tunnels).
  • If connection fails, try toggling airplane mode or switching between Flow/Digicel APN.
  • Some configs may require specific APN settings — check carrier recommendations.
  • HTTP Injector requires Android 4.1+ or a compatible Android device.

How SNI Bypass Works

Understanding the technology behind network tunneling

1

TLS Handshake

Your client initiates a TLS connection, sending the Server Name Indication (SNI) in plaintext.

2

SNI Inspection

ISPs inspect the SNI field to identify and block access to specific websites or services.

3

SNI Spoofing

We set the SNI to a whitelisted domain while routing traffic to the actual target through a proxy.

4

Connection Established

The ISP sees an allowed domain while your traffic reaches the intended destination securely.
